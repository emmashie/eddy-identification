The code in this repository is used for applying eddy identification and tracking of surf zone eddies using simulated vorticity. The simulated data used to perform eddy tracking is found here: https://doi.org/10.17603/ds2-esth-hg97


eddy_id.py : script to apply eddy tracking to simulated vorticity fields 

eddy_composite.py : script to generate data to create eddy composites from identified eddies 

eddy_tracking.py : script to track identified eddies from one time step to the next 

paper_figures.py : script to generate figures in Nuss et al XXXX
